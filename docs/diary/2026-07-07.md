### Context
Follow-up to the 2026-07-06 solo sharpen. Kjell and I walked the five open

### Human Inputs to Remember
- Grace window: 14 days TOTAL, 7 before and 7 after `expires_at`, not 7+14.
- Per-document version snapshots are overkill for this app; "history" means
  history of documents you have opened, not versions of one document.
- Free forever with only nagging should be challenged: the free tier keeps
  full core functionality, but valuable conveniences may be licensed-only.
- Auto-renewing subscription is acceptable when paired with transparent copy:
  cancel anytime, access lasts until the paid period ends.

### Decisions Locked
1. Merchant of record: Polar primary, Paddle fallback. Never fall back onto
   Stripe Managed Payments while it is preview.
2. Interim issuer: Polar license-key API. One-time online activation at
   license entry; revalidation only inside the 14-day window around
   `expires_at` (7 before, 7 after); cached state otherwise; all calls
   non-blocking for document operations. Custom Ed25519 issuer postponed.
3. Pricing: single SKU, $20/yr, auto-renewing subscription, cancel anytime,
   graceful return to free tier at period end.
4. Trial: the shipped reminder cadence (save 10, then every 25, session
   scoped) IS the trial. No time-boxed trial.
5. Feature gating: "no feature lockout" amended to "no lockout of core
   document operations; licensed-only additive conveniences permitted."
   Never turn off anything users already had free.
6. First gated feature: Document Library (unlimited searchable history of
   opened/converted documents), lands with go-live. Free tier keeps the
   shipped recents + session restore. On expiry the library stays browsable
   but stops recording. Desktop-only code (Swift + desktop adapter), MIT
   core untouched.
7. Expiry degradation: licensed → grace (calm banner, everything works) →
   expired-reminder (reminders resume, gated features pause, core untouched
   forever, no data hostage).
8. Clock rollback: no hardening beyond the existing future-issued_at check;
   honest-user model.
9. Web app: strict silence about the paid Mac app in the hosted app UI even
   after go-live. GitHub README and doc2md.dev are the talking surfaces.
10. Issuer spec: amend, don't retire. Status becomes deferred/v2 contingency;
    Ed25519 contract stays as the Polar exit plan; shipped verifier (PR #110)
    stays in the app dormant.
