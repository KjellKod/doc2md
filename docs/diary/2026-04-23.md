## Security Notes

- All URL-scheme-handled bytes come from user-selected paths via `NSOpenPanel` with `.startAccessingSecurityScopedResource`. No user-controlled URLs reach `fetch(importUrl)` — `importUrl` is constructed server-side by Swift with a 128-bit random token.
- Import route does no filesystem lookups by token. Unknown tokens return 404. Known tokens match against the single-slot active import.
- Save-target validation is three layers: NSSavePanel filter, ShellBridge check, FileStore defensive validator. The XCTest locks the innermost layer.
- `ShellError.localizedDescription` from `Data(contentsOf:)` can include file paths — user-selected, so low-impact, but noted for future redaction.
