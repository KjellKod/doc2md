## Quest: restart-new-doc_2026-04-28__2028

Outcome: complete. Made "start a new document" repeatable from any state with a shared dirty-safe reset path, a persistent hosted toolbar New button, and Mac Cmd+N parity.

Context:

- User explicitly opened with "in a worktree branch make this happen, solo mode or just do it?" — that is consent for solo mode and worktree workspace upfront, not a question to bounce back.
