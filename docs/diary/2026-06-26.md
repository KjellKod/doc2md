## Changes Made
- New `src/components/preview/tableCheckbox.ts`: `convertTableCellCheckboxes()`
  rewrites task-list checkbox markers inside GFM table cells to ☐ (U+2610) /
  ☑ (U+2611). Root cause: GFM task-list syntax only renders as a checkbox at
  the start of a list item, never inside a table cell.
- Wired into `convertClipboardPasteToMarkdown()` (`pasteToMarkdown.ts`) via a
  `pasteConversion()` helper so every markdown-producing paste branch runs it.
- Tests: repro in `pasteToMarkdown.test.ts` (red→green, unchanged by the fix)
  plus 13 unit tests in `tableCheckbox.test.ts`.
- Hardening after Dexter's review: bullet-or-whole-cell rule (no prose `[x]`
  corruption), fence char+length tracking, 0–3-space block recognition,
  inline-code skip, pipe-required delimiter rows.
- Branch `fix/table-cell-checkbox-glyph`, two commits, PR #191 (ready for review).
