# 2026-03-31 — LinkedIn Unicode Preview Quest

## Context
Quest `linkedin-unicode-preview_2026-03-31__2103` completed in solo mode. The work added a LinkedIn-oriented Unicode/plain-text preview branch to the existing preview panel without altering the normal Markdown preview or edit flow.

## Human Inputs to Remember
- User wanted this framed as LinkedIn formatting, not as a generic Unicode gimmick.
- User explicitly rejected a download/export path. The feature had to remain a preview/view option only.
- User wanted refusal instead of degraded mixed rendering when tables, HTML, or unsuitable constructs appeared.
- User specifically called out the isolation requirement: a bug in LinkedIn preview must not interfere with normal edit or normal preview.

- Guarded LinkedIn parsing so it runs only when the LinkedIn mode is selected.
