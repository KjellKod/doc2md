# doc2md Baseline Reference Outputs

This directory contains checked-in reference outputs generated by the current
`doc2md` CLI. They are kept for benchmark comparison and reviewer inspection,
not as casual output dumps.

To regenerate outputs for comparison:

```bash
node packages/core/bin/doc2md.js <input> -o <output-dir>
```

Example:

```bash
node packages/core/bin/doc2md.js \
  examples/API_Rate_Limiting_Design.docx \
  examples/Sprint_Metrics_Q1_2026.xlsx \
  -o ./examples/output/doc2md-baseline
```
